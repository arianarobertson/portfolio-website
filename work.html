Pokedex app

This project involves building a small web application using HTML, CSS, and JavaScript that retrieves data from an external API and allows users to view detailed information about individual data points. For this project, the technology I used was Pokémon API, CSS, JavaScript, ESLint, Prettier, and GitHub Pages.
